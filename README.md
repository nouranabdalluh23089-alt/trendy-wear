TrendyWear
Description:
TrendyWear is a modern and comfortable women's clothing website. Users can browse products by category and see details about each item. The website is responsive and works on desktops, tablets, and mobile devices.

Features:
- Display products with images and descriptions
- Categorized (Dresses, Pants, Tops)
- Easy navigation
- Responsive design

Technologies Used:
- HTML
- CSS
- JavaScript
- Node.js 

How to Run:
1. Open Terminal / Command Prompt
2. Navigate to the project folder
